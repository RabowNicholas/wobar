# CHANGE LOG — glass_orb

Hyperrealistic floating glass sphere composited over background video.
GLSL pixel shader — full sphere refraction (Snell's law, two-surface trace), Fresnel, chromatic aberration, Blinn-Phong specular. No 3D render pipeline.

- move_001: base_glass_orb COMP — moviefilein_bg → null_bg → glsl_orb (sphere refraction shader) → null_out; ctrl_orb CHOP for live parameter control
- [untracked — saved in .tox]: shader realism pass — scene reflection on outer glass surface (Fresnel²-weighted reflected UV sampling), caustic pool beneath orb (shadow core + bright crescent offset opposite light direction), saturation/contrast boost on refracted image, reduced background embedding (0.97), orb repositioned to horizon (center_y=0.47) to eliminate frosted gray-sky interior

- move_004: moviefileout_export — null_out → moviefileout; audio wired to base_audio/audio_in (stereo WAV); output /wobar/exports/glass_orb.mp4 @ 60fps sRGB
- move_005: 3 motes inside the orb — `base_motes` child COMP renders 3 instanced white spheres on perpendicular orbits via Script CHOP bridge (0.30/0.40/0.50 Hz, R=0.5 unit space); orthographic camera at z=4, ortho width 3.0; bloom = blurTOP(70px) → levelTOP(brightness 3.5) → compositeTOP(add) over render; `glsl_orb_pixel` patched to sample input[2] at refracted UVs (uvR/uvG/uvB) so motes pick up the glass refraction + chromatic aberration. Motion driven by `time_src` constantCHOP (expr=`absTime.seconds`) wired into pos_motes input — script CHOP needs an upstream time-dependent input to recook each frame. Tightened orbit R from 0.5 → 0.25. IT cistern reference, Act 1/2 cusp.
- move_006: removed specular highlights — both primary Blinn-Phong `spec` (was driven by ctrl_orb.spec_mult/spec_power) and hardcoded secondary `spec2` cut from `glsl_orb_pixel`. Orb now has no surface shine; motes carry all bright accents. ctrl_orb spec channels remain but are unused.
- move_007: tilted orbits + N-body physics — fixed prior 2-of-3 orbits being edge-on to camera (read as oscillation) by tilting their planes 60°; then replaced kinematic orbits entirely with stateful gravitational sim (Velocity Verlet, softened pairwise gravity, central spring, damping, smooth escape pull-back, 4× substeps). State on the callback DAT's `.storage` (NOT the script CHOP — `me.storage` in scriptCHOP onCook = DAT, not the CHOP itself). Added `ctrl_motes` constantCHOP (10 knobs) for live tuning. Initial seed = tilted orbit positions + tangent velocities, so frame 1 of physics matches the prior look and only diverges as bodies interact.
- move_008: Lennard-Jones repulsion to prevent collapse — bodies were drifting into a tight cluster from gravity + damping. Added F_rep = repulsion_k · ((min_dist/r)^4 − 1) at close range using TRUE r (not softened) so equilibrium sits at r=min_dist. Two new ctrl_motes channels (min_dist=0.18, repulsion_k=0.40). Also reverted a partial bloom restructure (blur_inner/level_inner/blur_outer/level_outer ops) that hit a dead end on compositeTOP only having 2 input slots — TODO: redo bloom stack with chained 2-input composites for the "ball of light" look.
- move_009: reverted move_008's bounce shell — wrong solution. The actual collapse cause was damping draining orbital energy. Removed the close-range repulsion code from `_accel`; set ctrl_motes damping = 0.0. Sim is now fully energy-conserving (gravity + central spring + outer pull-back) so motion persists indefinitely. Bodies can pass through each other freely; softening still caps gravity acceleration so close passes are smooth. min_dist / repulsion_k channels stay in ctrl_motes but are now no-ops.
- move_010: tightened orbit volume — bodies were riding the orb rim because seed R=0.25 + escape_r=2.0 let them swing far from origin. Reduced seed R to 0.12, escape_r to 0.5, restoring_k 0.30 → 0.45 for stiffer central confinement. Motes now move within the inner ~40% of the orb's projected disk with visible 3D depth motion.
- move_011: removed visual audio reactivity — `lvc_bg.saturationmult` was `1.0 + audio_energy*0.7`, now hard 1.0. `glsl_orb.vec2valuew` (glow_energy uniform) was `audio_energy`, now hard 0.0 (kills warm inner glow). `moviefileout_export.audiochop` left as-is (export sound track only). Audio reactivity will be re-introduced into the motes physics later, not into the orb shader.
- move_012: removed central restoring spring (restoring_k 0.45 → 0). The "collapse" was actually phase-synced harmonic oscillation: equal masses + equal spring constant → all 3 bodies oscillated through origin at the same 9.4s period. Removing the spring leaves only mutual gravity + outer pull-back wall, giving true 3-body dynamics with no central sync.
- move_013: star-like bloom redesign — sphere shrunk to 0.008 (essentially a point), 3-stage Gaussian bloom chain (inner sharp/bright, mid medium, outer wide-faint) chained via 2-input compositeTOPs (the corrected pattern after move_008's failed 3-input attempt). Material warmed (1.0, 0.95, 0.78) for stellar tint. Tried anisotropic blur for cross-flare diffraction spikes — didn't produce sharp rays, just stretched gaussians; reverted. True IT-cistern spikes would need a custom GLSL shader (deferred).
