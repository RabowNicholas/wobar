# Phase 3 — Final Recommendation

## The decisions locked in
- **Anchor: video platforms (TikTok + Instagram Reels).** Daily community-engagement energy goes here. Visualizer content native to TK/IG.
- **Foundation: SoundCloud.** Every track uploads to SC. Clean profile. No active community-engagement work beyond what happens naturally. The 16-release plan IS the SC credibility play.
- **Cadence: biweekly Fridays, 16 releases through 12/31** (5/29 → 12/25).
- **MUR ships 2026-05-29.** T-21 distributor deadline is Friday 2026-05-08.
- **Brand constraint: no face-of-artist content.** Visualizers carry video presence. The mirror is the artist.

## Tool Stack — what to spend on, when, and why

### Tier 1 — set up BEFORE MUR ships (next 3 weeks)
| Tool | Monthly cost | Status | Why |
|------|-------------|--------|-----|
| DistroKid | ~$2/mo annualized | Already have | Distribution to all DSPs. Free pre-save feature included. |
| SoundCloud Artist Pro | ~$8.25/mo ($99/yr) | Verify tier — possibly upgrade | 100% distribution royalties (Nov 2025 change), unlimited uploads. Load-bearing for the SC foundation strategy. |
| Spotify for Artists | Free | Claim if not done | Required for editorial pitches. Pitch MUR by T-14 (5/15). |
| Viberate for Artists | $2.99/mo | Add immediately | Cross-platform analytics. At $3/mo this is a no-brainer — Phase 1 showed the doc had this priced 13x too high. |
| TikTok + Meta Business Suite | Free | Use natively | Scheduling, analytics, posting. Don't pay for Buffer at this stage. |
| Canva (free tier) | Free | Use as-is | Single artwork per release. Don't need Pro. |

**Tier 1 total: ~$11/mo** (most of which you're already spending). Pre-MUR spend is essentially unchanged.

### Tier 2 — layer in across MUR + next 2 releases (Months 1–2)
| Tool | Cost | When to add | Why |
|------|-----|-------------|-----|
| Hypeddit Pro ($9.99/mo) OR DistroKid's free pre-save | $0–10/mo | Decide before MUR T-14 | Pre-save landing page. Phase 2 showed only 1 of 6 comps uses Hypeddit — not essential. Start with DistroKid's free pre-save; upgrade to Hypeddit only if pre-save conversion becomes a bottleneck. |
| SubmitHub credits | $30–50 per release (variable) | First credit batch before MUR T-14 | Target Electric Hawk's "Fresh Hunts" playlist specifically. Don't broad-cast — be surgical. ~5 curators per release. |

**Tier 2 incremental: $30–60/release variable + $0–10/mo recurring.**

### Tier 3 — defer until proven need (Month 3+)
| Tool | Why deferred |
|------|------------|
| MailerLite | **Phase 2 showed zero comp artists visibly use email at this stage.** Defer until ~5K monthly listeners or when a real fan-conversion bottleneck appears. The pre-save email opt-in from DistroKid/Hypeddit gives you a starter list at zero extra cost. |
| Bandzoogle | $11/mo Lite. Add when actively pursuing festival applications or press requiring an EPK. Not before. |
| Feature.fm | Smart links / pre-save. Overlaps with Hypeddit and DistroKid's free tools. Add only if specific need surfaces. |
| Buffer | Phase 1 showed $6/channel reality breaks the doc's math. TikTok + Meta native schedulers cover the same functions. Add Buffer or Metricool only if scheduling becomes a real bottleneck. |
| Groover | EU-focused. Low priority for SLC-based scene strategy. Revisit if expanding to EU touring. |

### Skip entirely for Wobar's stage
- **Kaiber AI** — you have TouchDesigner, you're a TD power user. Kaiber is for artists without your tooling.
- **Canva Pro** — free Canva is sufficient for single artwork per release.
- **PlaylistPush** — $250–1000 per campaign. Way over budget, premature for your save rate data.
- **Substack / Patreon** — community monetization tools for established artists.
- **MagicLinks / linkfire alternatives** — Hypeddit or DistroKid covers the same job.

**Recommendation: start at Lean for MUR, add Hypeddit at Release #2 if needed, decide on Bandzoogle/MailerLite around Release #6 based on actual traction.**

This is dramatically less than the original AI doc's $50/$100/$200 tier framing — which assumed Kaiber, Canva Pro, Buffer at distorted pricing, MailerLite as Tier 1, and other tools that Phase 2 evidence says don't apply at your stage.

## What changes in the release packet

### Section 3 (Release Execution) — rebuild

The current Section 3 has 4 sparse posting moments (T-14, T-7, Day 0, T+7). Nick flagged this as undersized. With video-anchored model + 5 visualizers per release + 0–2 live videos = 10–14 platform-posts per release window. Section 3 needs an explicit posting cadence sub-table.

**Proposed Section 3 structure:**

**3A — Distribution Setup (T-21, unchanged)**
- Distributor upload + Friday release date set
- Spotify for Artists editorial pitch
- Pre-save page (DistroKid free OR Hypeddit Pro)

**3B — Posting Cadence (T-14 → T+7, NEW)**
A date-by-date posting grid. For each post moment: date, content asset, platforms (TK + IG same day, OR alternating), done checkbox.

Default: same-day cross-post (TK + IG simultaneous). 5 unique visualizers + 1–2 live videos = 6–7 post moments × 2 platforms = 12–14 drops across 21 days = post every 3 days roughly. Day 0 is the release day big push.

**3C — Day 0 Release Mechanics**
- Release day post (hook clip primary)
- Network ask (save + follow + add to playlist)
- (Conduit Radio removed per Nick's note 2026-05-06)

**3D — Review (T+7, unchanged)**
- Section 4 metrics review
- Repost top clip with new caption
- Spotify Marquee TRIGGER ONLY if save rate >20%

### Press strategy — major rewrite

Original doc said: Mixmag, Dancing Astronaut, FUXWITHIT, Electric Hawk, The Untz tiered approach.

Phase 2 evidence: **FUXWITHIT covered 5 of 6 comp artists. EDM Identity is second. The other outlets are aspirational at this stage.**

Concrete approach:
- **FUXWITHIT — guest mix submission.** Submission process isn't public. DM the editor on Instagram. This is the single highest-leverage press relationship for Wobar's stage.
- **EDM Identity — track premiere pitches.** Less competitive than FUXWITHIT. Worth a shot per release.
- **The Untz — premiere pitches.** Festival-bass focused. Aligns with brand.
- **SubmitHub** — surgical use to target Electric Hawk's Fresh Hunts playlist + 3–4 other bass curators per release. Don't broad-cast.
- **Skip:** Mixmag, Dancing Astronaut, Magnetic Magazine until 25K+ monthly listeners.

### Strategic add — IP flips library

Phase 2 surfaced: **3 of 6 comp artists broke through on flips of recognizable IP** (Mport/Beatles, AYCH/Mac Miller, Kaiyo/Tove Lo+Kendrick+Zedd). The 16-release plan currently doesn't include flips. This is worth opening as a separate loop.

A flip strategy could fit Wobar specifically by:
- Flipping tracks already adjacent to psychedelic/transformation territory (Tame Impala, Tycho, Mac Miller's introspective work)
- Releasing flips as Channels-bucket drops (per WOBAR_CONTENT.md) — bypasses the full packet, builds catalog
- Flips often get higher initial discovery from people searching the original

Not for Phase 3 to resolve. Flag as new loop in next session.
